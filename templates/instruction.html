Instrukcja korzystania z api:

Wszystkie hasła zapisywane w bazie danych są po uprzednim zahashowaniu

Do skorzystania z API należy użyć api key, który zostaje wygenerowany każdemu użytkownikowi po rejestracji

Aby sprawdzić swój api key należy zalogować się na stronie

GET

API serwisu znajduje się pod adresem 6185.pythonanywhere.com/api/endpoint/?api_key=TWOJ_API_KEY.

Lista wszystkich użytkowników 6185.pythonanywhere.com/api/users?api_key=TWOJ_API_KEY

Wyszukać konkretnego użytkownika można poprzez dodanie id, username lub emaila do query api/users/?query=CO_CHCEMY_WYSZUKAC&api_key=TWOJ_API_KEY

Po id: 6185.pythonanywhere.com/api/users/?query=1&api_key=TWOJ_API_KEY

lub po username/emailu: 6185.pythonanywhere.com/api/users/?query=test&api_key=TWOJ_API_KEY


POST

Dodać użytkownika można poprzez POST, wzór danych:

{ "id": ' ',

"email": "EMAIL UZYTKOWNIKA"

"username": "NAZWA UZYTKOWNIKA"

"password": "HASLO" }


PUT

Aktualizować dane użytkownika można za pomocą metody POST

Aktualizacje danych należy dokonać poprzez wskazanie id użytkownika w linku

Przykład linku do POST: 6185.pythonanywhere.com/api/users/?query=1&api_key=TWOJ_API_KEY

Do aktualizacji danych należy wykorzystać powyższy wzór danych

Można aktualizować pojedyńcze dane poprzez pominięcie innych w wzorze


DELETE

Usuwać dane użytkownika można za pomocą metody DELETE

Usunięcie użytkownika należy dokonać poprzez wskazanie id użytkownika w linku

Przykład linku do DELETE: 6185.pythonanywhere.com/api/users/?query=1&api_key=TWOJ_API_KEY
